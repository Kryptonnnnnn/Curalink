🚀 Curalink AI — Medical Research Assistant

An AI-powered full-stack web application that aggregates medical research data and generates intelligent summaries using LLMs.

🌐 Live Demo
Frontend: https://curalink-gules.vercel.app
Backend API: https://curalink-1-ltoq.onrender.com

🧠 Features

🔍 Search medical queries (disease, treatments, trials)
📄 Fetch research papers from PubMed and OpenAlex
🧪 Retrieve clinical trial data
🤖 Generate AI-powered summaries using Groq LLM
💾 Store user chat history (MongoDB)
⚡ Real-time response with retry handling
🌐 Fully deployed (Vercel + Render)

🏗️ Tech Stack

Frontend
React.js
Axios
Tailwind CSS

Backend
Node.js
Express.js
MongoDB (Mongoose)

AI & APIs
Groq API (LLaMA 3.1)
PubMed API
OpenAlex API
Clinical Trials API

Deployment
Vercel (Frontend)
Render (Backend)

🚀 Installation & Setup

1. Clone repo
git clone https://github.com/your-username/curalink.git
cd curalink

2. Backend setup
cd backend
npm install
npm run dev

3. Frontend setup
cd frontend
npm install
npm start
